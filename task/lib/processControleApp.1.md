# **Name processControleApp**


## **Purpose** 
this app developed to manage some process features like:  
1. list process  
2. send signal to process

## **Usage** 

### To use the app
> ./processControleApp 

### To See Help Menu
> ./processControleApp -h 

## **Signals Available** 
* SIGHUP
* SIGINT
* SIGQUIT
* SIGTRAP
* SIGABRT
* SIGKILL
* SIGTERM
* SIGCONT
* SIGSTOP

## See Also
> ps(1) , kill(1)
